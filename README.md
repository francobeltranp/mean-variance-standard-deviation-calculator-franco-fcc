# Mean-Variance-Standard Deviation Calculator

This is a freeCodeCamp Data Analysis with Python project.

## Description

A function that uses NumPy to calculate the mean, variance, standard deviation, max, min, and sum of the rows, columns, and elements in a 3x3 matrix.

## Usage

```python
import mean_var_std

result = mean_var_std.calculate([0,1,2,3,4,5,6,7,8])
print(result)
```

## Running Tests

